# Contributing to DNYF DevStudio

Thank you for your interest in contributing to DNYF DevStudio! 🎉

## 🌟 Ways to Contribute

- 🐛 Report bugs
- 💡 Suggest new features
- 📝 Improve documentation
- 🔧 Submit pull requests

## 🚀 Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/devstudio.git`
3. Create a branch: `git checkout -b feature/amazing-feature`
4. Make your changes
5. Test thoroughly
6. Commit: `git commit -m 'Add amazing feature'`
7. Push: `git push origin feature/amazing-feature`
8. Open a Pull Request

## 📋 Pull Request Guidelines

- Keep changes focused and atomic
- Write clear commit messages
- Add tests if applicable
- Update documentation
- Follow the existing code style
- Test on multiple browsers

## 🎨 Code Style

- Use ES6+ JavaScript features
- Follow React best practices
- Use functional components with hooks
- Keep components small and focused
- Write self-documenting code
- Add comments for complex logic

## 🧪 Testing

```bash
npm run dev    # Test in development
npm run build  # Test production build
npm run preview # Test production locally
```

## 📞 Questions?

Open an issue or contact the DNYFTECH team!

**Thank you for contributing! 💙**
